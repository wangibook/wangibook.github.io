# vue-admin-template

提示

这是一个极简的 vue admin 管理后台。它只包含了 Element UI & axios & iconfont & permission control & lint，这些搭建后台必要的东西。

线上地址：
vue-admin-template

# 安装

# 克隆项目
git clone https://github.com/wangibook/vue-admin-template.git

# 进入项目目录
cd vue-admin-template

# 安装依赖
npm install

# 启动服务
npm run dev
