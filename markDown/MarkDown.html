# Markdown基本语法

Markdown是一种纯文本格式的标记语言。通过简单的标记语法，它可以使普通文本内容具有一定的格式。

# 标题

在想要设置为标题的文字前面加#来表示 一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。

注：标准语法一般在#后跟个空格再写文字

示例：

# 一级标题  
### 三级标题  
##### 五级标题

# 效果如下：

# 这是一级标题

# 这是三级标题

# 这是五级标题

# 文字

  • 要加粗的文字左右分别用两个*号包起来
    这是加粗的文字

  • 要倾斜的文字左右分别用一个*号包起来
    这是倾斜的文字

  • 要倾斜和加粗的文字左右分别用三个*号包起来
    这是斜体加粗的文字

  • 要加删除线的文字左右分别用两个~~号包起来
    这是加删除线的文字 示例：

**这是加粗的文字**
*这是倾斜的文字*`
***这是斜体加粗的文字***
~~这是加删除线的文字~~

# 效果如下：

这是加粗的文字
这是倾斜的文字
这是斜体加粗的文字
这是加删除线的文字

# 段落

Markdown 段落没有特殊的格式，直接编写文字就好，段落的换行是使用两个以上空格加上回车。
当然也可以在段落后面使用一个空行来表示重新开始一个段落。

# 提示

::: tip 提示
this is a tip
:::

::: warning 注意
this is a tip
:::

::: danger 警告
this is a tip
:::

# 效果如下：

提示

this is a tip

注意

this is a tip

警告

this is a tip
